
# 跨尺度课题三

## 1 数据集

为验证所提方法在多尺度场景三维重建任务中的有效性与适应性，构建了涵盖“大场景-中场景-小场景”三类典型尺度场景的数据集，包含真实采集数据与合成数据，并辅以人工标注与多模态信息对齐，具体数据规模、类型及用途如下：
### 1.1 大场景数据集

大场景数据集聚焦城市级、区域级室外环境的三维重建任务，包含真实采集的无人机航拍图像与激光雷达点云数据，覆盖建筑群、道路、植被等复杂地物。
数据集共收录大尺度场景样本5组，每组包含RGB图像800–2000张（分辨率≥4K），覆盖面积1–3 km²。数据来源包括UrbanScene3D公开数据集及本项目自采集数据。所有图像附带相机内外参、曝光时间、GPS/IMU信息。

### 1.2 中场景数据集

中场景数据集面向室内环境或局部建筑结构的三维重建，强调复杂结构与材质细节的还原能力。
数据集共收录中尺度场景样本3组，涵盖车辆、实验室、住房等典型中型场景。每组包含多视角RGB图像200–500张（分辨率4K）。数据来源包括Mip-NeRF 360、3DGS自带数据集tank and temple数据集等室内外场景及采集数据。

### 1.3 小场景数据集

小场景数据集聚焦以物体为中心的精细化重建任务，涵盖静态物体、文物、工业零件等小尺度目标。
数据集共收录小尺度样本4组，包含多视角图像序列（每序列50–150张图像，分辨率4K）。样本覆盖多种材质类型（漫反射、镜面、透明）与几何复杂度（简单几何体至雕刻细节）。数据来源包括NeRF-Synthetic、DTU数据集及自采数据集。
